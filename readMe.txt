관련된 폴더명과 파일명은 작업자가 임의생성합니다.

작업순서
1. 결과화면을 보고 데스크탑용 html, css 코드를 생성합니다.
2. 결과화면을 보고 데스크탑용 와이어프레임과 
   정보구조도를 만듭니다.
3. 결과화면을 참조하여 태블릿용 와이어프레임을 만듭니다.
   (콘텐츠 내용이 작은 관계로 정보구조도는 만들지 않습니다.)
4. 태블릿용 와이어프레임을 참조하여 
   태블릿용 html, css 화면을 구현합니다.
5. 결과화면을 참조하여 스마트폰용 와이어프레임을 만듭니다.
   (콘텐츠 내용이 작은 관계로 정보구조도는 만들지 않습니다.)
6. 스마트폰용 와이어프레임을 참조하여 
   스마트폰용 html, css 화면을 구현합니다.

요약
1. 결과화면을 보고 
	와이어프레임, 정보구조도(태스크플로우 제작안함) 제작
	데탑용 html, css 결과 소스 구현
2. 요안도라 결과 소스 참조해
	향기나는 꽃집 태블릿용 와이어프레임 제작
	태블릿용 html, css 결과 소스 구현
3. 요안도르 결과 소서 참조해
	향기나는 꽃집 스마트폰용 와이어프레임 제작
	스마트폰용 html, css 결과 소스 구현
4. 작업완료 후 닷홈 FTP에 특정 폴더 만들고 업로드 후 데스크탑 스마트폰에서 결과확인
	특정폴더명 : html/rwd/ex/
